# 🥦 Omnifood — AI-Powered Food Subscription

> *A healthy meal delivered to your door, every single day.*

Omnifood is a fully responsive landing page for a fictional AI-powered food subscription service. Built with modern HTML and CSS, it features smooth animations, mobile-first design, and a clean, contemporary UI — demonstrating real-world front-end development skills.

---

## 🌐 Live Demo

👉 [View Live](https://sanmithsj1103.github.io/omni-food/)

---

## ✨ Features

- **Fully Responsive** — seamlessly adapts to desktops, tablets, and mobile devices
- **Smooth Scrolling & Animations** — CSS transitions and scroll-based effects for a polished feel
- **Mobile Navigation** — hamburger menu for smaller screens
- **Sections include:**
  - Hero / Call-to-Action
  - How It Works (step-by-step)
  - Featured Meals
  - Meal Plan Options & Diets
  - Testimonials
  - Pricing Plans
  - Sign-Up / Contact Form
  - Footer with social links

## 🚀 Getting Started

### Prerequisites

All you need is a modern web browser — no build tools or dependencies required.

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/sanmithsj1103/omni-food.git

# 2. Navigate into the project folder
cd omni-food

# 3. Open index.html in your browser
open index.html
```

Or simply drag and drop `index.html` into your browser.

## 🤖 About the Project

Omnifood is a technology-first food company with a focus on consumer well-being through healthy eating. The AI algorithm generates a **100% personalized weekly meal plan** based on your diet and preferences — and then delivers it right to your door.

This project was built as part of the **"Build Responsive Real-World Websites with HTML and CSS"** course on MERN stack development, showcasing practical front-end development skills and best practices.
